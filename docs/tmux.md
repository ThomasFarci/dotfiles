## Requirements
- tmux
- tpm (the Tmux Plugin Manager)

## Notes

When installing new plugins or using this config file for the first time, run
```sh 
tmux source ~/.tmux.conf
```
if tmux is already running to make it use the updated config file, then press prefix + I (capital i) to install plugins. You can press prefix + U (capital u) to update plugins. It will ask which to update, you can choose `all`.
